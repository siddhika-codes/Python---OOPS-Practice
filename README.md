# Python---OOPS-Practice
Ye repo mera OOP (Object-Oriented Programming) practice ka collection hai, jo maine Python seekhte waqt banaya hai.

Mene OOP k chaar pillar kiye h.
OOP ke 4 pillars practice kiye
- Abstraction
- Encapsulation
- Inheritance
- Polymorphism
Har pillar ke folder mein 3-3 mini tasks hain jo maine khud solve kiye practice ke liye.
Abhi seekh rahi hoon — aage in sabko combine karke ek chhota project banane ka plan hai.
